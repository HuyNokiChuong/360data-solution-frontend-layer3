# 09. User Management - Quan Ly Nguoi Dung

## 1. Muc tieu module
- Moi nguoi dung vao workspace.
- Quan ly role, status, note, tags.
- Gan quyen granular theo dashboard.

## 2. Quy trinh moi user moi
1. Vao `Users`.
2. Bam `Invite`.
3. Nhap ten, email, role.
4. Them note/tags neu can.
5. (Tuy chon) gan quyen theo tung dashboard.
6. Gui invite.

## 3. Chinh sua user da co
1. Chon user trong bang.
2. Bam icon edit.
3. Sua role/note/tags.
4. Cap nhat quyen granular dashboard.
5. Luu thay doi.

## 4. Khoa/mo khoa va xoa user
- Khoa/mo khoa: bam icon lock/unlock.
- Xoa user: bam icon trash.
- Khi xoa, nen kiem tra dashboard share lien quan.

## 5. Rule domain
- User moi phai dung email corporate.
- Domain email user moi phai trung domain workspace.
