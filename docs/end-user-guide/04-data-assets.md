# 04. Data Assets - Quan Ly Bang Du Lieu

## 1. Muc tieu module
- Quan ly danh sach bang/sheet da dong bo.
- Bat/tat bang cho cac module phan tich.
- Preview du lieu thuc te truoc khi dua vao bao cao.

## 2. Chuc nang chinh
- Tim kiem theo ten table/dataset.
- Loc theo connection.
- Sort theo ten hoac row count.
- Bat/tat trang thai `Active/Paused`.
- Xoa le hoac xoa hang loat.
- Xem truoc du lieu (`Data Preview`) co phan trang.

## 3. Quy trinh thao tac khuyen nghi
1. Vao `Data Assets`.
2. Loc theo connection dang dung de giam nhieu.
3. Tat (`Paused`) cac bang khong phuc vu dashboard/AI.
4. Bam `Preview` de kiem tra schema va sample data.
5. Neu bang sai nguon hoac du lieu rac, xoa khoi registry.

## 4. Luu y nghiep vu
- Xoa table o day chi go bo khoi he thong, khong xoa bang goc tren data source.
- Ask AI va Dashboard Studio nen dung table `Active` de tranh context sai.
- Preview BigQuery can token hop le; neu het han can auth lai.

## 5. Loi thuong gap
- Preview ra rong: table co the chua dong bo xong hoac khong co row.
- Bat/tat status that bai: kiem tra ket noi backend/token.
- Bulk delete sai: nen loc + chon check box ky truoc khi xoa.
