# 08. AI Settings - Cau Hinh API Key

## 1. Muc tieu module
- Cau hinh API key cho cac provider AI.
- Kiem tra ket noi key.
- Mo khoa model tuong ung trong he thong.

## 2. Provider ho tro
- Google Gemini
- OpenAI (GPT)
- Anthropic (Claude)

## 3. Quy trinh thao tac
1. Vao `AI Settings`.
2. Nhap API key tung provider.
3. Bam `Test Connection` de kiem tra key.
4. Bam `Save Neural Configuration` de luu.
5. Vao lai Ask AI/Dashboard de chon model.

## 4. Luu y bao mat
- Khong chia se API key qua chat/email cong khai.
- Nen xoay key dinh ky.
- Neu key bi leak va bi khoa, tao key moi va cap nhat ngay.

## 5. Loi thuong gap
- Test key fail: key sai, het han, hoac bi revoke.
- Khong thay model trong dropdown: chua cau hinh key provider do.
